# Proyecto Final: LabCar 

Este proyecto consiste en replicar el diseño tanto para desktop como para mobiles.
Además se pide la localización del usuario y marcarlo en el mapa con la API de GoogleMaps.

Aqui una pequeña introducción de la empresa:

LabCar es una empresa de taxis que te permite pedir un taxi cuando quieras y en donde quieras.
Tenemos una gran cantidad de conductores por todo el departamento.
Se ha comprobado que LabCar tiene las tarifas más bajas del mercado.
Sabemos que tu tiempo es dinero, por lo que si nuestro servicio se demora en llegar más de lo establecido te daremos un 10% de
descuento en tu viaje.
